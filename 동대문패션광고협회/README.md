## 2. 2023년 기능경기대회 `동대문패션광고협회`
  + ### 과제 설명
    + 이 대회는 일반적인 기능경기대회가 아닌 `(사)국제올림픽선수협회`가 주최했던 민간대회입니다.
    + 대회 진행중 인터넷 검색이 허용된 대신, 과제가 미리 공개되지 않았으며, 화상 회의 `ZOOM`을 통해 진행되었습니다.
    + 2023년 6월에 진행했던 대회 과제입니다. 추가로 처음으로 수상했던 대회입니다.
  + ### 완성도
    + 프론트 엔드와 백 엔드 양측 일부 완성
    + 사용한 언어
      + `HTML`, `CSS`
      + `Java Script`
      + `PHP`, `MYSQL`
  + ### 어려웠던 부분
    + A모듈(HTML, CSS)에서 마우스 커서의 위치에 따라서 요소가 변형되는 문제가 있었는대, 그 당시에 개발하지 못했다. 나중에 찾아본 결과 일반적인 노가다 작업이였다.
    + A모듈(HTML, CSS)에서 서브페이지에 상품 카드에 카드를 이미지로 직접 만들어서 출력을 시켰어야 하는데 시간을 많이 잡아먹었다.
  + ### 쉬웠던 부분
    + 인터넷 검색이 허용이였던지라 몆몆의 시간이 많이걸리는 과제 이외의 다른 문제들은 수월했다.
